Popularity based - A popularity-based recommendation system is a non-personalized, generalized filtering approach that suggests items to users based on current trends, high ratings, or high purchase/view counts. It is based on a formula.
Content-Based - Recommends items based on the features (metadata) of items the user has liked previously (e.g., genre, director, description). It does not require other users' data. It is based on similarity of user data. 
Collaborative Filtering (CF): Predicts user preferences based on the actions and behaviors of similar users, or by finding similarities between items. It find the most similar item based on the how the user and other people liked it using coordinate system based upon number of users.
Hybrid Recommendation Systems: Combines popularity, collaborative filtering and content-based approaches to overcome the limitations of each, such as the "cold start" problem. Examples include using weighted, switching, or feature combination methods. It works by giving weightage to the recommended items by each method and then finally calculating the best item.
